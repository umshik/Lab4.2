# Лабораторная работа 4.2 — Циклический двусвязный список

Оконное приложение на Python с графическим интерфейсом (Tkinter) для работы с циклическим двусвязным списком. Список реализован в виде четырёх библиотек на разных языках программирования.

## Библиотеки

| Библиотека | Язык | Файл |
|---|---|---|
| C | C | cDLL.dll |
| C++ STL | C++ (std::list) | C++DLL.dll |
| Pascal | Pascal | PascalDLL.dll |
| Python | Python | PythonDLL.py |

## Функциональность

- Добавление элемента в конец списка
- Добавление элемента перед i-м
- Добавление элемента после i-го
- Удаление i-го элемента
- Удаление элемента перед i-м
- Удаление элемента после i-го
- Поиск по числовому полю
- Поиск по символьному полю
- Определение длины списка
- Очистка списка

## Запуск

1. Убедитесь что все DLL-файлы находятся в одной папке с `app.py`
2. Установите Python 3.x
3. Запустите `app.py`
```bash
python app.py
```

## Структура проекта
```
├── app.py          # Основное приложение (GUI)
├── PythonDLL.py    # Библиотека на Python
├── cDLL.dll        # Библиотека на C
├── C++DLL.dll      # Библиотека на C++ STL
├── PascalDLL.dll   # Библиотека на Pascal
└── C++DLL.cpp      # Исходный код C++ STL библиотеки
```
